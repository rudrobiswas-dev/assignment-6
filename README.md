### DigiTools - Digital Tools Marketplace

# Project Description
DigiTools is a modern digital tools marketplace where users can explore, purchase, and manage premium tools like AI software, design assets, and productivity solutions—all in one place. The platform provides a smooth and interactive shopping experience with a clean UI.

### Technologies Used
⚛️ React.js
⚡ Vite
🎨 Tailwind CSS
🔔 React Toastify

# Features
Add to Cart & Remove Items with real-time updates
Toast Notifications for user actions (add, remove, checkout)
Toggle View between Products and Cart
